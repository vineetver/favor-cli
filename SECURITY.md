# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

1. **Do not** open a public issue
2. Email [vineetxverma@gmail.com](mailto:vineetxverma@gmail.com) with details
3. Include steps to reproduce if possible
4. Expect a response within 72 hours
